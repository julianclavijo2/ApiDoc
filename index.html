Introduction

ShekinAPI is a product made by SHEKINAPI that includes a set of web services for online hotel data extraction and distribution partners. Using this API, the partners are able to access to availability, rates, property content, product details for worldwide hotel companies, as well as to send reservation and cancellation transactions. With ShekinAPI, partners can access to travel data from any existing travel services supplier. The cost of developing an Interface for new distributors using this API is highly reduced.
