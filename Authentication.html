Authentication

The access codes to API provided by SHEKINAPI make the server allow the authentication in the API. The credentials must be sent at the beginning with the following parameters:

To authenticate with API use the following JSON code:

Parameter Description Optional
user "usser" no
pass 6e9552c9bd8e61c8f277c21220160233 no
token 37027ef916f3c798faa9cff051dd3577 no

                                    "POS":{
                                    "user":"usser",
                                    "pass":"6e9552c9bd8e61c8f277c21220160233",
                                    "token":"37027ef916f3c798faa9cff051dd3577"
                                   }
