Errors

This section shows the errors that may appear in the services in order to identify the wrong operation. This API use the following error code:

Error Code Meaning
100 Access Denied -- The information of the authentication is not valid
200 The service "Cancellation Policies" was not executed
201 The service "Enhanced" was not executed
300 An error has ocurred
301 Availability not Found
600 Booking code was not created
700 It was not possible to load the information in session to generate the order
800 There was a mistake creating the order
900 Service not available, contact the administrator
1000 The maximum amount of records are 1000
1001 The quantity must be greater than 0
